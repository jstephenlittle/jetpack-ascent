# Sprites

This folder contains all sprite sheets and image assets for Jetpack Ascent.

## Organization

### Character Sprites
- `jetty.png` - Player character sprite sheet (all animations)
- `enemies_robots.png` - Robot enemy sprite sheet (Roller Bot, Hover Drone, DropBot)

### Level Themes
- `tiles_space.png` - Space Tower platforms and decorations
- `tiles_gothic.png` - Gothic Tower platforms and decorations
- `tiles_business.png` - Business Tower platforms and decorations

### UI Elements
- `ui.png` - HUD icons, bars, and UI elements
- `powerups.png` - Power-up sprites (Fuel Cell, Shield, Mega Boost, Extra Life)

## Specifications

All sprites follow the specifications in `graphics.doc`:
- Base tile size: 32×32 px
- Player character: 32×32 px
- Enemies: 24×24 px (Roller Bot, DropBot), 28×20 px (Hover Drone)
- Power-ups: 16×16 px
- 2px padding between frames in sprite sheets

## Placeholder Phase

During initial development, colored rectangles and KAPLAY shapes are used as placeholders. Actual pixel art sprites will be created per `graphics.doc` specifications.
