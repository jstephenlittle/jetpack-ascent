# Audio

This folder contains all sound effects and music for Jetpack Ascent.

## Organization

### Sound Effects (sfx/)
- Jetpack thrust
- Platform landing
- Power-up collection
- Enemy collision/death
- Player death
- Menu interactions
- Breakaway platform crumble
- Bounce pad launch
- Doorway transition

### Music (music/)
- Main menu theme
- Level 1 (Space Tower) background music
- Level 2 (Gothic Tower) background music
- Level 3 (Business Tower) background music
- Victory theme
- Game over theme

## Format
- Format: `.mp3` or `.ogg` (KAPLAY supports both)
- Sample rate: 44.1kHz recommended
- Quality: Balance between file size and quality for web delivery

## Status
Audio implementation is deferred to Issue #36 (future enhancement). This folder is prepared for when audio assets are ready to be integrated.
